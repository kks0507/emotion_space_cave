# 선행연구 조사 — 단계적 루브릭 (O2)

조사일 2026-06-17 / 목적: 학생 정서 자유텍스트를 0~N단계로 단계화하고 각 단계에 시스템 행동(일반응답/주의관찰/관리자알림·개입)을 매핑하는 루브릭의 학술 근거 확보. **임상 자살평가가 아니라 학업중단(자퇴·휴학) 예방·학적 유지가 목적.**

## 축 1. 학업중단 조기경보(EWS)·정서적 이탈·개입 임계값
- **EWS 임계값 트레이드오프**: 낮은 임계값=오탐↑(과잉 식별), 높은 임계값=미탐↑(놓침). "이탈 확률 50% 초과 지점"을 경계로 권고. (NCES Forum Guide to EWS 2019; UNICEF EWS 2018)
- **IAFREE 모델**(가장 직접 참고): 전통적 ABC(출결·행동·성적) 지표가 정서적 단절을 놓친다는 비판 → 5개 관계 차원 + 자기보고 정서항목 추가. 예: *"나는 슬프거나 우울하고 이것이 학교 중단을 생각하게 한다"*. 7점 리커트 + IRT 보정. (Vasconcelos et al. 2023, Frontiers in Psychology 14:1189283)
- **MTSS 3계층**(행동 매핑 표준): Tier1 보편(전체·예방) / Tier2 표적(소그룹·데이터기반) / Tier3 집중(개별·고강도). → 우리의 일반응답/주의관찰/관리자알림에 1:1 대응.
- **한국 정책**: 학업중단 숙려제 위기요건 = 무단결석 연속 7일/누적 30일, 자퇴 징후 등(명시적 수치 임계값). KEDI Brief 5호(2026.4): 2024년 학업중단 ~5만명, "사후처방 → 조기탐지·조기개입" 전환 제언, **"수업 이해도(주관) → 심리·정서 위기로 전이"** 경로 제시. (※ 수치는 언론보도 기반, 원문 대조 권장)

## 축 2. 정서/부정 정서 강도 측정 척도
- **PANAS**: 부정 정서 10문항 × 0~4 → 합 0~40 연속 강도. 검증된 표준(α 0.84~0.87).
- **SemEval-2018 Task 1 (Affect in Tweets)**: EI-reg(0~1 회귀) + **EI-oc(0/1/2/3 서열 분류)**. "부정 정서를 이산 단계로 분류"가 확립된 NLP 표준 태스크. → 우리 0~N 단계화의 직접 모델.

## 축 3. 텍스트 기반 위기 신호 단계화 (단계→행동 매핑 원형)
- **C-SSRS**: 자살관념 5단계 서열(소망→비특이 능동→방법→약한 의도→구체적 계획+의도) + 강도 별도 측정 + triage(저/중→재평가, 고위험→전문가·에스컬레이션). **차용 원리 3가지**: ① 수동→능동·구체로 깊어지는 서열, ② "종류"와 "강도" 분리, ③ 상위 단계일수록 자동처리 대신 사람에게 에스컬레이션. **단, 임상 라벨 복제 금지 — "학업 이탈 의사 서열"로 재정의.**
- 텍스트만으로 우울·자살위험 단계 분류 가능(PMC9170939). 이탈 신호는 탈락 한참 전 출현(arXiv 2602.00021).

## 축 4. 한국어 감정/부정어 자원
- **KNU 한국어 감성사전**(군산대): 5단계 극성(매우부정~매우긍정, 통상 -2~+2), 14,843항목(부정 쪽 두터움: 부정5,029/매우부정4,797). → 부정 강도 가중에 직접 활용. (※ 라이선스 확인 필요)
- **AI Hub 감성대화 말뭉치**: 60종 세부감정, 27만 문장, 우울 관련 시나리오 포함.
- 국내 청소년 위기언어 텍스트마이닝 실증(PMC10686066).

## 루브릭 설계 직접 시사점
1. **3계층 행동 매핑 = MTSS(Tier1/2/3) 차용.** 단계 수는 0~4(5단계) 권장(PANAS·EI-oc·C-SSRS 모두 4~5단계).
2. **"종류"와 "강도" 2축 분리.** 부정 정서 강도(KNU·NLP) ⟂ 이탈 의사 구체성(막연한 무기력→"그만두고 싶다"→구체적 자퇴 언급). max(두 축)으로 단계 결정 → 미탐 방지.
3. **임계값은 명시·검증가능. 최상위 단계는 자동응답 대신 사람으로 에스컬레이션.**
4. **객관지표만으론 정서적 이탈 못 잡음(IAFREE·KEDI) → 자유텍스트 정서추적의 포지셔닝 명분.**
5. **한국어: KNU 사전(강도) + AI Hub 감성대화(분류) 앙상블.**
6. **임상 척도는 패턴만 차용, 라벨은 학업중단 맥락으로 재정의. 진짜 자살·자해 신호엔 별도 안전 트랙(외부 위기상담 연계).**
